O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux.

Os fundadores do github foram Tom Preston-Werner, Chris Wanstrath, Scott Chacon, P. J. Hyett. Em 2018 a Microsoft compra GitHub por US$ 7,5 bilhões.

Pode se trabalhar com o github de diversas formas, como pelo VSCode, pelo software oficial do git e ferramentas de terceiros como gitkraken.
